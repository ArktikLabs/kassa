# Code of Conduct

The Kassa project has adopted the **Contributor Covenant, version 2.1** as its Code of Conduct. The canonical text of the Covenant (including the Our Pledge, Our Standards, Enforcement Responsibilities, Scope, Enforcement, and Enforcement Guidelines sections) is published at:

- <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

All contributors, maintainers, reviewers, and other participants in Kassa project spaces — including the code repository, issue trackers, pull requests, and any other official Kassa communication channel — are expected to abide by the Covenant.

## Scope

This Code of Conduct applies within all project spaces, and it also applies when an individual is officially representing the project or its community in public spaces.

## Enforcement

Instances of abusive, harassing, or otherwise unacceptable behavior may be reported to the project maintainers at:

- **Email:** conduct@kassa.example

All complaints will be reviewed and investigated promptly and fairly. Project maintainers are obligated to respect the privacy and security of the reporter of any incident. Enforcement actions will follow the Community Impact Guidelines described in the Covenant.

## Attribution

This Code of Conduct is adapted from the Contributor Covenant, version 2.1, available at the link above. Community Impact Guidelines were inspired by Mozilla's code of conduct enforcement ladder.

For answers to common questions about this Code of Conduct, see the FAQ at <https://www.contributor-covenant.org/faq>. Translations are available at <https://www.contributor-covenant.org/translations>.
